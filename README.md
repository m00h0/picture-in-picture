## 📖 About the Project <a name="about-project"></a>

This project allows users to select a media stream from their screen and display it in a Picture-in-Picture (PiP) window. It is built using HTML, CSS, and JavaScript.

<p align="right">(<a href="#readme-top">back to top</a>)</p>

### 🛠 Built With <a name="built-with"></a>
-  HTML, CSS, JavaScript

<p align="right">(<a href="#readme-top">back to top</a>)</p>

#### Tech Stack <a name="the-tech-stack"></a>

- **Languages**: HTML, CSS, JavaScript
- **Libraries**: None
- **Frameworks**: None
- **Tools**: Webpack, Babel

<p align="right">(<a href="#readme-top">back to top</a>)</p>

#### Key Features <a name="key-features"></a>

- **Picture-in-Picture Mode**: Allows users to select a media stream and display it in a Picture-in-Picture window.
- **User-Friendly Interface**: Simple and intuitive interface for easy interaction.
- **Cross-Browser Support**: Works on modern web browsers that support the Picture-in-Picture API.
- **Responsive Design**: Ensures a seamless experience across different devices and screen sizes.

## 💻 Getting Started <a name="getting-started"></a>

To get a local copy up and running, follow these simple steps.

### Setup <a name="setup"></a>

Clone the repository:

```sh
git clone https://github.com/your-username/picture-in-picture.git

```

### Prerequisites <a name="prerequisites"></a>

- Node.js and npm installed
- A modern web browser that supports Picture-in

### Install <a name="install"></a>

Install dependencies:

```sh
npm install
```
### Usage <a name="usage"></a>

Start the development server:

```sh
npm start
```

### Run Tests <a name="run-test"></a>

To run tests, execute(Coming soon):

```sh
npm test
```

### Deployment <a name="deployment"></a>

The project is deployed and accessible at the following link:

[Live Demo](https://m00h0.github.io/picture-in-picture/)

To deploy this project yourself, follow these steps:

1. **Build the project**:
   ```sh
   npm run build
   ```

2. Deploy to GitHub Pages:

    - Push your code to a GitHub repository.
    - Go to the repository settings.
    - Scroll down to the "GitHub Pages" section.
    - Select the branch you want to deploy from (usually main or gh-pages) and click "Save".

3. Access your deployed project:

    - Once deployed, you can access your project via the URL provided by GitHub Pages.
